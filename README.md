# Recursive binary search - VB.NET
This project demonstrates a **recursive binary search algorithm** implemented using **Visual Basic .NET** in a **Windows forms desktop application**

## Features
-Recursive binary search
-Windows Forms UI
-Input validation

## How to Run
1. Clone the repository
2. Open "Recursive binary search.sln" in Visual Studio
3. Click **Start**

## Technology
-Visual Basic .NET
-Windows Forms
-.NET Framework
